# Lineage barcode sequence filtering, processing and clone calling.
## Contents of this directory:
- Functions for UMI-based filtering of lineage barcode sequences on endogenous genes - endo_target_filtering.R
- Functions for filtering of lineage barcode sequences on transgenic targets - transgene_scar_filtering_functions.R
- Lineage barcode probabilities for selected genes - Gene_scar_probs.csv
- Functions for pairing up alleles and grouping cells based on individual endogenous genes - endo_per_target_clone_calling.R  
- Function for calling clones based on all target genes - clone_def_all_targets.R
- Full example workflow with inline outputs - full_clone_calling_multi_bAllos4_tums.ipynb 
- Example intermediate outputs including plots - director 'output'
